## Deployment Information
- Instructions on how to deploy the app: 
  1. Ensure you have R and Shiny installed.
  2. Clone the repository.
  3. Run `app.R` in RStudio or use `shiny::runApp()` command.

## Updated File Structure
- `app.R`                 : Main application file.
- `data/`                 : Contains datasets used by the app.
- `www/`                  : Static files such as CSS and JavaScript.
- `presentation/`         : Folder containing presentation materials.
- `README.md`             : This readme file.
